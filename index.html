Some of my Work

Bus - Ticket Booking


Bus-Ticket Booking is a web app focused on booking a bus seat in advance. It also allows the user to revise travel history using unique PNR. Booked seat can also be canceled before the travelling date using the PNR.



JAVA Digital Library


JAVA Digital Library is a JAVA project using AWT and SWING. Main purpose of this project is to showing the book's name and displaying the contents of the book. MongoDB is used to store data of the user and book details.

NodeJS - MongoDB CRUD Operations


This is a simple web app to demonstrate MongoDB CRUD operations on NodeJS using web app. This allows users to Sign-Up or Sign-In. After the user enters into his account it will show the user's details in dashboard. The user can edit his details or delete his account.
